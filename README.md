# Grup 9
El projecte consisteix a elaborar una aplicació informàtica per gestionar el control aeri d'un aeroport, a partir dels conceptes apresos a classe, mitjançant classes, funcions, fitxers, llistes i més.
## **Versio 1**
A la versió 1, l'objectiu principal és crear una base per gestionar la informació d'un aeroport, a partir de codis ICAO, latitud, longitud i l'espai Schengen.

Video: https://youtu.be/GRKCngWhAcE

## **Versio 2**
A la versió 2, s'incorporarà el tràfic aeri amb l'arribada d'avions a l'aeroport de Barcelona de diferents bandes del món; s'utilitzaran dades d'aerolínies, aeroports d'origen i hores d'arribada.

Video: https://youtu.be/F_fUuaCxXK0

## **Versio 3**
A la versió 3, es modelarà l'estructura de l'aeroport de Barcelona amb les seves terminals, àrees d'embarcament i portes. Es dissenyarà un codi que trobi i assigni les portes als avions que vagin arribant a Barcelona.

Video: https://youtu.be/fJk7VDCZZo8

## **Versio 4**
A la versió 4, s'incorporaran els enlairaments a la versió 3 i s'haurà de dissenyar un sistema que vagi assignant i alliberant portes dels avions que arribin o se'n vagin.

Video: https://youtu.be/r3rcr3W_RaU
